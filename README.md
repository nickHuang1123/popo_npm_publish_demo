# @nick-huang/popo

一個不知道在幹甚麼的套件，裡面有Vue3、Element-plus、Sass、Vite。

## Install
```bash
# 安裝前須先安裝 Element-plus
yarn add @nick-huang/popo
```

## Use
```javascript
import { SButton, SInput } from '@nick-huang/popo';
import '@nick-huang/popo/style';
```
